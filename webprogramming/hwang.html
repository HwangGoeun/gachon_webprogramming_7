What is Autonomous driving?




This refers to a car that can be driven by itself without the operation of the driver or passengers.

Autonomous driving cars can be classified from level 0 to 5 according to the classification of automation stages, and level 3 is viewed as autonomous driving cars.

Software related to autonomous driving

ROS : Robot Operating System

The Robot Operating System (ROS) is a set of software libraries and tools that help you build robotic applications.
It has everything you need for robotics projects, and it's open source.

C++

It is a C language-based programming language, which is necessary for ROS programming.

Python

Dynamic typing universal programming language, required for ROS programming, like C++.

Linux

Because ROS software works primarily in Linux environments, Linux knowledge is required.
